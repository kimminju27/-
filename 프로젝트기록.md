## 새 글 추가하는 방법
1. posts/sample.html 복사 → 새 파일명으로 저장 (예: posts/etf-guide.html)
2. 제목, 내용 수정
3. index.html 포스트 카드 섹션에 카드 추가
4. 터미널에서 아래 명령어 실행:
   git add .
   git commit -m "새 글 추가: 글제목"
   git push
5. Vercel이 자동으로 사이트에 반영
